Designing with frameworks

During this assignment I gained the knowledge of some CSS frameworks like Bootstrap, Pico and Tailwind.
After having experimenting a little in a sandbox file, I implemented some Bootstrap elements to the assignment.

Details

In order to create a navigation menu, I decided to use Bootstrap. By adding a few utilities classes, I was able to implement a burger menu that is easy to understand and use.

I have also used Bootstrap to layout the webpage with grid and flexes and make it responsive. By using the col and row utilities I could set how many columns or rows a section could take and define that for different screen sizes.
