What I learned

During this theme I expanded my knowledge about good design conventions and have applied them with the help of focused assignments. Some of the assignments are presented in a list down below.

Assignments

13C.01.01
Fonts and UI components

The focus of this assignment was to lean how to work with fonts by pairing them and how to better showcase UI components.

13C.01.02
Working with colors and animations

Through this assignment, I dove deep into the use of colors and how to use animations with MotionOne and javascript.

13C.02.02
Designing with frameworks

In this assignment I experimented by using CSS frameworks to make the project responsive.

Conclusions

During this theme I reached a deeper understanding of good design conventions, layout patterns, the importance of fonts, colors and how to best work with UI elements, cards and animations, with or without the use of CSS frameworks.
