13C.01.01
Fonts and UI components

By doing this assignment I learned the importance of pairing fonts and how it can change the whole experience and vibe of a website.
It's good convention to use a sans-serif font for the parts of text we want to make sure people read and it's important to think about line height, letter spacing and font hierarchy to enhance the value of the website and make it accessible to everyone.
A serif font can be used to express a particular feeling or vibe.

Details

For the fonts I have chose two google fonts by pairing a sans-serif with a serif font. This makes the reading clear but still keeps it interesting and gives the website a more user friendly tone. I find this combination of fonts works well, so I applied it through the rest of the assignments.

For the cards, I tried to apply the good conventions we were tought: media, text with hierarchy, rounded corners, slight shadows.
